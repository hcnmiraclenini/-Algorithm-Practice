## 📖 项目简介

本项目是一个系统性的算法与数据结构学习代码库，包含：

- **基础算法**：数学计算、简单逻辑问题
- **数据结构**：线性表、树、图、哈希表等经典数据结构
- **算法实现**：排序、查找、字符串匹配等常用算法
- **面向对象编程**：C++ 类、继承、多态等特性
- **实际应用**：银行系统、身份证验证等综合项目

## 🔍 主要分类

### 01_基础算法
包含基础数学问题和简单逻辑算法，如：
- 模运算与队列操作
- 字符串数字识别
- 水仙花数、闰年判断
- 大数处理等

### 02_排序算法
实现了多种经典排序算法：
- **O(n²) 算法**：冒泡排序、选择排序、插入排序
- **O(n log n) 算法**：快速排序、归并排序、堆排序
- **其他**：基数排序、稳定排序等

### 03_查找算法
- 顺序查找
- 二分查找（折半查找）

### 04_字符串算法
字符串处理和匹配相关算法：
- KMP 字符串匹配算法
- 字符串最长前后缀
- 字符串重复子串查找
- 字符串输入输出操作

### 05_树结构
完整的树结构实现和应用：
- **二叉树**：构建、遍历、查找、镜像反转等
- **平衡树**：AVL 树、平衡因子计算
- **特殊树**：哈夫曼树、B 树
- **树操作**：前驱后继、节点距离、叶子节点统计等

### 06_图算法
图论相关算法：
- **搜索算法**：深度优先搜索（DFS）、广度优先搜索（BFS）
- **最短路径**：Dijkstra 算法
- **最小生成树**：Kruskal 算法
- **其他**：拓扑排序、图的邻接矩阵表示等

### 07_栈与队列
栈和队列的应用：
- 栈的基本操作
- 中缀转后缀表达式（波兰式、逆波兰式）
- 括号匹配
- 栈实现迷宫求解
- 行编辑器实现

### 08_链表
链表数据结构实现：
- 单链表操作（STL 实现）
- 链表节点交换
- 有序链表合并
- 宿舍管理系统（链表版）

### 09_哈希表
哈希表及其冲突处理方法：
- 链地址法
- 线性探测再散列
- 二次探测再散列

### 10_动态规划与数学
- 全排列生成
- 零钱兑换问题
- 中位数查找
- 单调栈应用
- 高精度计算

### 11_面向对象编程
C++ 面向对象编程实践：
- **类与对象**：构造函数、析构函数、拷贝构造
- **继承与多态**：继承机制、虚函数
- **友元函数**：友元类的使用
- **STL 应用**：set、list、vector 等容器
- **运算符重载**：自定义类型运算符

### 12_系统应用
综合应用系统项目：
- 银行账户管理系统
- 身份证验证系统（15位/18位转换）
- 电视机管理系统
- 旅程信用卡积分系统
- 学生生日差值计算

### 13_实验作业
课程实验和作业代码：
- 期末模拟考试
- 计算机组成实验
- 迎新赛题目

## 🚀 使用说明

### 环境要求
- Visual Studio 2017 或更高版本
- 支持 C++11 标准的编译器

### 运行代码
1. 打开 Visual Studio
2. 选择对应的 `.sln` 解决方案文件
3. 按 `F5` 编译并运行，或 `Ctrl+F5` 运行（不调试）

### 项目特点
- 每个算法都有独立的 Visual Studio 项目
- 代码结构清晰，包含注释
- 适合学习和参考使用
